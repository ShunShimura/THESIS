\begin{frame}[noframenumbering]{提案手法1：スライスを跨いだIDの割り振り}
    \begin{itemize}
        \item 口頭：
        \begin{itemize}
            \item スライスを跨いだIDを割り振る
            \item 縦向きの動きに追従することができる，推定のもと
        \end{itemize}
        \item 図：跨いだReIDの図（全スライドと対応するように）
        \item 図：深さ方向に追跡を行うことで，どれが同じ個体由来であるかを知ることができる
    \end{itemize}
\end{frame}
